## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

Navigate to the specific folder and install dependencies:

```bash
# For Node.js basics
cd 01-node-basics
npm install

# For Express.js
cd 02-node-js-with-express
npm install
```

### Running the Server

```bash
# From 01-node-basics folder
node server.js

# Server will run on http://localhost:9000
```
